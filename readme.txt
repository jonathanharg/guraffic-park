# Guraffic Park

A PyGame OpenGL 3D scene.

## Install

This project works with **Python 3.11 & 3.12** on **Windows**.

Install the dependencies (recommended to do this in a virtual environment with `python -m venv .venv`)

```bash
pip install -r requirements.txt
```

## Running

Run the project by executing the module with `.` or running `main_scene.py`.

```bash
cd guraffic-park && python .
```
